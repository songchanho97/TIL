1. while문에 의한 문장의 반복

1.1 반복문의 이해와 while문 
- 반복문이란?
  - 하나 이상의 문장을 두 번 이상 반복 실행하기 위해서 구성하는 문장 


```c
int main(void)
{
  int num = 0;
  while(num < 5)
  {
    printf("Hello world! %d \n", num);
    num++;
  }
  return 0;
}  
```


2. do ~ while 문

```c
do 
  {
    printf("Hello world! \n");
    num++;
  }while(num<3);
```

- 반복조건을 반복문의 마지막에 진행하는 형태이기 때문에 최소한 1회는 반복영역을 실행하게 된다. 


```c
int main(void)
{
	int total=0, num=0;
	do
	{
		  printf("정수입력(0 to quit): ");
		  scanf("%d", &num);
		  total += num;	
	}while(num!= 0);
	
		printf("합계: %d \n", total);
	
	return 0;
}
```
